# SATELINK UNIFIED MASTER REFERENCE

## What is Satelink?

Satelink is a decentralized infrastructure network (DePIN).

It converts routers, servers, GPUs, and edge devices into infrastructure providers.

The network executes real workloads and settles revenue in USDT.
