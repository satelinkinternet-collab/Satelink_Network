# Contributing to Satelink Network

Welcome! We appreciate your interest in contributing to the Satelink Network.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Commit your changes with descriptive messages.
4. Open a pull request against the `develop` branch.

## Branch Strategy
- `main` -> production
- `staging` -> pre-release
- `develop` -> integration (PRs here)

## Documentation
Please refer to the `docs/` directory for architecture, economics, and developer guides.
