```javascript
const aggregate = await client.query(`
    SELECT
        COUNT(*)::integer AS event_count,
        COALESCE(SUM(amount_usdt), 0)::numeric AS total_revenue_usdt
```

**ISSUE:** Sums ALL revenue_events_v2 rows for epoch. No validation of:
